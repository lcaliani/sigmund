# Sigmund
Aplicação desktop de gerenciamento de clínica de pscologia para estudo da ferramenta Electron

## Pré-requisitos

- [`NodeJS >=14`](https://nodejs.org/en/)
- [`NPM >= 6`](https://nodejs.org/en/)

## Instalação

```sh
npm i
```

## Uso

#### Inicializar aplicação

```sh
npm start
```

#### Criar executável

```sh
npm run make
```
